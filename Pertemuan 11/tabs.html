Bahasa program yang digunakan terutama untuk mengembangkan halaman web dinamis. Bahasa yang sangat populer untuk menambahkan fitur interaktif untuk halaman web dan aplikasi, memvalidasi data formulir, memanggil API (Application Programming Interface), pengembangan game web, dan lain-lain.
